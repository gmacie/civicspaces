== README

This app was born out of the Atlanta City Govathon in 2013.

CivicSpaces is a site to help active citizens, artists and nonprofits find unused

spaces to host their events and display thier creative work. We serve as a 

conduit, connectiong oweners of empty spaces with people who want to fill them for

a shorty-term project, event or activity

* Ruby version 2.0.0
* Rails version 4.0.0
